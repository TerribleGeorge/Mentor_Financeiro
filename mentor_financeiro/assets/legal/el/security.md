# Ασφάλεια - Mentor Financeiro

**Τελευταία ενημέρωση:** 19 Μαΐου 2026

Αυτή η σελίδα εξηγεί τις πρακτικές ασφαλείας Mentor Financeiro σε απλή γλώσσα. Συμπληρώνει την Πολιτική Απορρήτου και τους Όρους Χρήσης.

## 1. Δέσμευση ασφάλειας

Το Mentor Financeiro αναπτύχθηκε για να βοηθάει με χρηματοοικονομικούς οργανισμούς χωρίς να ζητάτε άμεση πρόσβαση στον τραπεζικό λογαριασμό, χωρίς να ζητάτε τραπεζικό κωδικό πρόσβασης και χωρίς να λειτουργεί ως τράπεζα, μεσίτης ή τρόπος πληρωμής.

Η εφαρμογή μπορεί να αποθηκεύσει οικονομικά δεδομένα που έχετε καταχωρίσει ή προσδιορίζονται από προαιρετικές λειτουργίες, γι' αυτό αντιμετωπίζουμε την ασφάλεια ως κεντρικό μέρος του προϊόντος.

## 2. Τι δεν κάνει η εφαρμογή

- Δεν ζητά κωδικό πρόσβασης τράπεζας, κάρτα ή ψηφιακό πορτοφόλι.
- Δεν έχει πρόσβαση στον τραπεζικό σας λογαριασμό.
- Δεν κινεί χρήματα.
- Δεν πραγματοποιεί πληρωμές, Pix, μεταφορές ή επενδύσεις στο όνομά σας.
- Δεν ζητά κωδικό ασφαλείας κάρτας.
- Δεν ζητά τον τραπεζικό κωδικό OTP.
- Δεν πουλά τα προσωπικά σας δεδομένα.

## 3. Τεχνικά μέτρα που χρησιμοποιούνται

Ανάλογα με τον πόρο που χρησιμοποιείται, η εφαρμογή μπορεί να χρησιμοποιήσει:

- Είσοδος από Firebase Authentication.
- Cloud Firestore με δεδομένα που σχετίζονται με τον πιστοποιημένο χρήστη.
- Κανόνες ασφαλείας για περιορισμό της πρόσβασης στα δεδομένα του ίδιου του χρήστη.
- Επικοινωνία κρυπτογραφημένη από τα SDK και τις υπηρεσίες που χρησιμοποιούνται.
- Firebase Crashlytics για τον εντοπισμό τεχνικών βλαβών και τη διόρθωση προβλημάτων.
- Τα δικαιώματα Android ζητούνται μόνο όταν σχετίζονται με λειτουργίες εφαρμογής.
- Τοπική αποθήκευση για προτιμήσεις, θέμα, προσωρινή μνήμη και δεδομένα συσκευής.

## 4. Παρακολούθηση ειδοποιήσεων

Η παρακολούθηση ειδοποιήσεων είναι προαιρετική και απαιτεί μη αυτόματη ενεργοποίηση στο Android.

Όταν είναι ενεργοποιημένη, η εφαρμογή μπορεί να διαβάζει υποστηριζόμενες ειδοποιήσεις για να αναγνωρίζει δαπάνες, αγορές, πληρωμές, συνδρομές και μεταφορές. Στόχος είναι να μειωθεί η χειρωνακτική εργασία κατά την καταγραφή των δαπανών.

Για να μειώσετε τους κινδύνους:

- Η εφαρμογή σάς ενημερώνει για τον λόγο της άδειας προτού σας κατευθύνει στις ρυθμίσεις.
- Η πρόσβαση μπορεί να ανακληθεί ανά πάσα στιγμή στις ρυθμίσεις Android.
- Το φίλτρο αναζητά συμβατά οικονομικά γεγονότα, όχι προσωπικές συνομιλίες.
- Η εφαρμογή προσπαθεί να αγνοήσει τους κωδικούς OTP, τους κωδικούς πρόσβασης, τους κωδικούς ασφαλείας και τα μηνύματα ελέγχου ταυτότητας.
- Η εφαρμογή δεν χρησιμοποιεί αυτή τη δυνατότητα για πρόσβαση σε τράπεζες, κάρτες ή ψηφιακά πορτοφόλια.

Ακόμα κι έτσι, οι ειδοποιήσεις μπορεί να περιέχουν ευαίσθητα δεδομένα. Χρησιμοποιήστε αυτήν τη δυνατότητα μόνο εάν συμφωνείτε με αυτόν τον τύπο επεξεργασίας.

## 5. Ευαίσθητα δικαιώματα

Η εφαρμογή μπορεί να ζητήσει:

- **Διαδίκτυο:** σύνδεση με Firebase, Google Play, API και διαδικτυακούς πόρους.
- **Ειδοποιήσεις:** αποστολή ειδοποιήσεων από την ίδια την εφαρμογή.
- **Τοποθεσία:** προσαρμογή περιεχομένου ή συγκρίσεις στη χώρα/περιοχή, όταν επιτρέπεται.
- **Πρόσβαση σε ειδοποιήσεις Android:** προαιρετική αναγνώριση δαπανών από συμβατές ειδοποιήσεις.
- **Αγορές εντός εφαρμογής:** Premium συνδρομή από Google Play.

Μπορείτε να ελέγξετε αυτά τα δικαιώματα στις ρυθμίσεις Android.
