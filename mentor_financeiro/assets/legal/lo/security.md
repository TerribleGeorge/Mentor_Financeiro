# ຄວາມປອດໄພ - Mentor Financeiro

**ອັບເດດຫຼ້າສຸດ:** ວັນທີ 19 ພຶດສະພາ 2026

ໜ້ານີ້ອະທິບາຍ Mentor Financeiro ການປະຕິບັດຄວາມປອດໄພໃນພາສາທີ່ກົງໄປກົງມາ. ມັນໃຫ້ສົມບູນກັບນະໂຍບາຍຄວາມເປັນສ່ວນຕົວແລະເງື່ອນໄຂການນໍາໃຊ້.

## 1. ຄໍາຫມັ້ນສັນຍາຄວາມປອດໄພ

Mentor Financeiro ໄດ້ຖືກພັດທະນາຂຶ້ນເພື່ອຊ່ວຍໃນການຈັດຕັ້ງທາງການເງິນ ໂດຍບໍ່ຕ້ອງຂໍການເຂົ້າເຖິງບັນຊີທະນາຄານໂດຍກົງ, ໂດຍບໍ່ມີການຮ້ອງຂໍລະຫັດຜ່ານທະນາຄານ ແລະ ໂດຍບໍ່ມີການເຮັດວຽກເປັນທະນາຄານ, ນາຍຫນ້າ ຫຼືວິທີການຊໍາລະເງິນ.

ແອັບດັ່ງກ່າວສາມາດເກັບຂໍ້ມູນການເງິນທີ່ທ່ານປ້ອນເຂົ້າ ຫຼື ລະບຸໄດ້ໂດຍຄຸນສົມບັດທາງເລືອກ, ເຊິ່ງແມ່ນເຫດຜົນທີ່ພວກເຮົາຖືຄວາມປອດໄພເປັນສ່ວນໜຶ່ງຂອງຜະລິດຕະພັນ.

## 2. ສິ່ງທີ່ແອັບຯບໍ່ໄດ້ເຮັດ

- ບໍ່​ໄດ້​ຮ້ອງ​ຂໍ​ໃຫ້​ລະ​ຫັດ​ຜ່ານ​ທະ​ນາ​ຄານ​, ບັດ​ຫຼື wallet ດິ​ຈິ​ຕອນ​.
- ບໍ່​ໄດ້​ເຂົ້າ​ເຖິງ​ບັນ​ຊີ​ທະ​ນາ​ຄານ​ຂອງ​ທ່ານ​.
- ບໍ່​ໄດ້​ຍ້າຍ​ເງິນ​.
- ບໍ່​ໄດ້​ເຮັດ​ໃຫ້​ການ​ຊໍາ​ລະ​ເງິນ​, Pix​, ການ​ໂອນ​ຫຼື​ການ​ລົງ​ທຶນ​ໃນ​ຊື່​ຂອງ​ທ່ານ​.
- ບໍ່​ໄດ້​ຮ້ອງ​ຂໍ​ລະ​ຫັດ​ຄວາມ​ປອດ​ໄພ​ຂອງ​ບັດ​.
- ບໍ່ຂໍລະຫັດທະນາຄານ OTP.
- ບໍ່​ໄດ້​ຂາຍ​ຂໍ້​ມູນ​ສ່ວນ​ບຸກ​ຄົນ​ຂອງ​ທ່ານ​.

## 3. ມາດຕະການດ້ານວິຊາການທີ່ນໍາໃຊ້

ອີງຕາມຊັບພະຍາກອນທີ່ໃຊ້, app ສາມາດນໍາໃຊ້:

- ເຂົ້າສູ່ລະບົບໂດຍ Firebase Authentication.
- Cloud Firestore ກັບຂໍ້ມູນທີ່ກ່ຽວຂ້ອງກັບຜູ້ໃຊ້ທີ່ຢືນຢັນແລ້ວ.
- ກົດລະບຽບຄວາມປອດໄພເພື່ອຈໍາກັດການເຂົ້າເຖິງຂໍ້ມູນຂອງຕົນເອງຂອງຜູ້ໃຊ້.
- ການສື່ສານທີ່ຖືກເຂົ້າລະຫັດໂດຍ SDKs ແລະບໍລິການທີ່ໃຊ້.
- Firebase Crashlytics ເພື່ອກໍານົດຄວາມຜິດທາງດ້ານວິຊາການແລະແກ້ໄຂບັນຫາ.
- Android ການ​ອະ​ນຸ​ຍາດ​ຮ້ອງ​ຂໍ​ພຽງ​ແຕ່​ໃນ​ເວ​ລາ​ທີ່​ກ່ຽວ​ຂ້ອງ​ກັບ​ຄຸນ​ນະ​ສົມ​ບັດ app​.
- ການ​ເກັບ​ຮັກ​ສາ​ທ້ອງ​ຖິ່ນ​ສໍາ​ລັບ​ຄວາມ​ມັກ​, ຫົວ​ຂໍ້​, ຖານ​ຄວາມ​ຈໍາ​ແລະ​ຂໍ້​ມູນ​ອຸ​ປະ​ກອນ​.

## 4. ການຕິດຕາມການແຈ້ງເຕືອນ

ການ​ຕິດ​ຕາມ​ກວດ​ກາ​ແຈ້ງ​ການ​ເປັນ​ທາງ​ເລືອກ​ແລະ​ຮຽກ​ຮ້ອງ​ໃຫ້​ມີ​ການ​ກະ​ຕຸ້ນ​ດ້ວຍ​ຕົນ​ເອງ​ໃນ Android​.

ເມື່ອເປີດໃຊ້ງານ, ແອັບສາມາດອ່ານການແຈ້ງເຕືອນທີ່ຮອງຮັບເພື່ອຮັບຮູ້ການໃຊ້ຈ່າຍ, ການຊື້, ການຈ່າຍເງິນ, ການສະໝັກໃຊ້ ແລະການໂອນເງິນ. ເປົ້າຫມາຍແມ່ນເພື່ອຫຼຸດຜ່ອນການເຮັດວຽກຄູ່ມືໃນເວລາທີ່ບັນທຶກຄ່າໃຊ້ຈ່າຍ.

ເພື່ອຫຼຸດຜ່ອນຄວາມສ່ຽງ:

- app ແຈ້ງ​ໃຫ້​ທ່ານ​ຮູ້​ເຫດ​ຜົນ​ສໍາ​ລັບ​ການ​ອະ​ນຸ​ຍາດ​ກ່ອນ​ທີ່​ຈະ​ນໍາ​ທ່ານ​ໄປ​ທີ່​ການ​ຕັ້ງ​ຄ່າ​.
- ສາມາດຖອດຖອນການເຂົ້າເຖິງໄດ້ທຸກເວລາໃນການຕັ້ງຄ່າ Android.
- ຕົວກອງຊອກຫາເຫດການທາງດ້ານການເງິນທີ່ເຂົ້າກັນໄດ້, ບໍ່ແມ່ນການສົນທະນາສ່ວນຕົວ.
- ແອັບຯພະຍາຍາມບໍ່ສົນໃຈລະຫັດ OTP, ລະຫັດຜ່ານ, ລະຫັດຄວາມປອດໄພ ແລະຂໍ້ຄວາມການຢືນຢັນ.
- ແອັບບໍ່ໄດ້ໃຊ້ຄຸນສົມບັດນີ້ເພື່ອເຂົ້າເຖິງທະນາຄານ, ບັດ ຫຼືກະເປົາເງິນດິຈິຕອນ.

ເຖິງແມ່ນວ່າ, ການແຈ້ງເຕືອນອາດມີຂໍ້ມູນທີ່ລະອຽດອ່ອນ. ໃຊ້ຄຸນສົມບັດນີ້ພຽງແຕ່ຖ້າທ່ານຕົກລົງກັບປະເພດຂອງການປຸງແຕ່ງນີ້ເທົ່ານັ້ນ.
