- สถานที่ตั้งเป็นทางเลือกและใช้เฉพาะเมื่อคุณอนุญาตเท่านั้น เช่น เพื่อปรับการเปรียบเทียบการลงทุนให้เหมาะกับบริบทของประเทศหรือท้องถิ่น
- คุณสมบัติการตรวจสอบการแจ้งเตือนเป็นตัวเลือก หากคุณเปิดใช้งานการเข้าถึงการแจ้งเตือนบน Android แอปจะสามารถอ่านชื่อและข้อความของการแจ้งเตือนที่รองรับเพื่อระบุการซื้อ การชำระเงิน และการโอน และเปลี่ยนสิ่งเหล่านี้ให้เป็นบันทึกทางการเงิน แอปไม่เข้าถึงบัญชีธนาคารของคุณและไม่ถามรหัสผ่าน
- รูปภาพที่คุณเลือกสำหรับการปรับเปลี่ยนในแบบของคุณสามารถจัดเก็บไว้ในอุปกรณ์ได้
- คุณสามารถขอลบบัญชีและข้อมูลผ่านทางอีเมลที่ระบุในนโยบายนี้

## 3. ข้อมูลที่เราอาจประมวลผล

เราอาจประมวลผลข้อมูลประเภทต่อไปนี้ ทั้งนี้ขึ้นอยู่กับวิธีที่คุณใช้แอปพลิเคชัน

### 3.1 ข้อมูลบัญชีและข้อมูลประจำตัว

- ชื่อที่ระบุในการลงทะเบียน
- อีเมล.
- Firebase (UID) ตัวระบุผู้ใช้
- วิธีการเข้าสู่ระบบที่ใช้
- รูปโปรไฟล์เมื่อได้รับจากผู้ให้บริการเข้าสู่ระบบ
- Premium สถานะการสมัครสมาชิกที่เกี่ยวข้องกับโปรไฟล์
- ข้อมูลทางเทคนิคที่จำเป็นในการรักษาเซสชันและซิงโครไนซ์ข้อมูล

### 3.2 ข้อมูลทางการเงินที่คุณป้อน

- ค่าใช้จ่าย รายจ่าย รายได้ หมวดหมู่ ประวัติการเงินและธุรกรรม
- เป้าหมาย ขีดจำกัด การวางแผน ความฝันทางการเงิน และความชอบ
- ข้อมูลที่ใช้ในกราฟ รายงาน การวิเคราะห์ และการให้คำปรึกษาอัตโนมัติภายในแอป
- ประวัตินักลงทุน วัตถุประสงค์ และคำตอบที่ให้ไว้ในแบบสอบถามหรือหน้าจอการวางแผน

ข้อมูลนี้อาจมีความละเอียดอ่อนในทางปฏิบัติ เพราะมันเผยให้เห็นแง่มุมต่างๆ ของชีวิตทางการเงินของคุณ ดังนั้นคุณควรลงทะเบียนเฉพาะข้อมูลที่คุณต้องการเก็บไว้ในแอปพลิเคชันเท่านั้น

### 3.3 ข้อมูลการสมัครสมาชิกและการซื้อ

- ผลิตภัณฑ์ที่ซื้อ สถานะการสมัครสมาชิก ระยะเวลา การต่ออายุ การกู้คืนการซื้อ และตัวระบุทางเทคนิคที่จำเป็นในการยืนยัน Premium
- การซื้อ การยกเลิก การต่ออายุ การคืนเงิน และการเรียกเก็บเงินจะดำเนินการโดย Google Play ตามบัญชี Google และกฎของร้านค้า

แอปไม่รวบรวมหรือจัดเก็บหมายเลขบัตร รหัสความปลอดภัยของบัตร หรือรายละเอียดการชำระเงินผ่านธนาคาร

### 3.4 ข้อมูลการใช้งาน การวินิจฉัย และประสิทธิภาพ- เหตุการณ์การใช้งานและการโต้ตอบในแอป เช่น หน้าจอที่เข้าถึง ทรัพยากรที่ใช้ และโฟลว์ที่เสร็จสมบูรณ์
- ข้อมูลรวมหรือข้อมูลทางเทคนิคเพื่อทำความเข้าใจความเสถียร ประสิทธิภาพ และการปรับปรุงของผลิตภัณฑ์
- รายงานข้อผิดพลาดและการวินิจฉัยโดย Firebase Crashlytics เมื่อเกี่ยวข้อง รวมถึงข้อมูลทางเทคนิคของอุปกรณ์ เวอร์ชันแอป ระบบปฏิบัติการ บันทึกข้อผิดพลาด และตัวระบุผู้ใช้เมื่อกำหนดค่า

### 3.5 ข้อมูลการแจ้งเตือนแบบพุช

หากคุณอนุญาตการแจ้งเตือนจากแอป เราสามารถจัดการ:

- โทเค็น Firebase Cloud Messaging (FCM)
- การตั้งค่าที่เกี่ยวข้องกับการรับการแจ้งเตือน
- หัวข้อหรือส่วนที่จำเป็นสำหรับการส่งการสื่อสารจากแอปพลิเคชันเอง

คุณสามารถควบคุมว่าจะอนุญาตการแจ้งเตือนหรือไม่ในการตั้งค่า Android

### 3.6 ข้อมูลการแจ้งเตือนอ่านใน Android

การตรวจสอบการแจ้งเตือนเป็นคุณสมบัติเสริม ใช้งานได้เฉพาะเมื่อคุณเปิดใช้งานการเข้าถึงการแจ้งเตือนด้วยตนเองในการตั้งค่า Android

เมื่อเปิดใช้งาน แอปจะสามารถรับชื่อ ข้อความ ชื่อแพ็กเกจแอปต้นทาง เวลา และข้อมูลเมตาทางเทคนิคจากการแจ้งเตือนที่แสดงบนอุปกรณ์ของคุณ วัตถุประสงค์คือเพื่อระบุการแจ้งเตือนที่เข้ากันได้กับค่าใช้จ่าย การซื้อ การชำระเงิน การสมัครสมาชิก และการโอนเพื่อสร้างบันทึกทางการเงิน

มาตรการที่สำคัญของฟีเจอร์นี้:

- แอปไม่สามารถเข้าถึงบัญชีธนาคารของคุณ
- แอพนี้ไม่ได้ล็อกอินเข้าสู่ธนาคาร บัตร หรือกระเป๋าเงินดิจิทัล
- แอปไม่ถามรหัสผ่านธนาคาร
- แอปใช้ตัวกรองเพื่อหลีกเลี่ยงการจับรหัสความปลอดภัย OTP รหัสผ่าน และข้อความการตรวจสอบสิทธิ์
- จุดเน้นของตัวกรองคือการระบุการซื้อที่เข้ากันได้ การชำระเงิน การโอนเงินที่ส่ง และความเคลื่อนไหวทางการเงิน
- คุณสามารถปิดการใช้งานคุณสมบัติในแอปหรือเพิกถอนการอนุญาตได้ในการตั้งค่า Android

แม้จะมีตัวกรอง การแจ้งเตือนก็อาจมีข้อมูลที่ละเอียดอ่อนได้ เปิดใช้งานคุณสมบัตินี้เฉพาะเมื่อคุณเข้าใจและยอมรับการประมวลผลประเภทนี้

### 3.7 ที่ตั้ง

แอปสามารถขอตำแหน่งโดยประมาณหรือตำแหน่งที่แน่นอนได้ ตามการอนุญาตที่ได้รับจาก Android เพื่อปรับการเปรียบเทียบ เนื้อหา หรือบริบทการลงทุนให้เข้ากับประเทศ/ภูมิภาค

ไม่จำเป็นต้องระบุตำแหน่งสำหรับฟีเจอร์ทั้งหมด หากคุณปฏิเสธการอนุญาต การเปรียบเทียบหรือการปรับแต่งบางอย่างอาจถูกจำกัด

### 3.8 รูปภาพและไฟล์ในเครื่อง

เมื่อคุณเลือกรูปภาพเพื่อปรับแต่งแอปด้วยสายตา รูปภาพนั้นจะถูกจัดเก็บไว้ในอุปกรณ์ เว้นแต่จะมีการนำไปใช้เป็นการเฉพาะในอนาคต ภาพการปรับแต่งส่วนบุคคลเหล่านี้จะไม่ถูกส่งไปยังเซิร์ฟเวอร์ของเรา

### 3.9 การโฆษณา

แอปอาจแสดงโฆษณาผ่าน Google AdMob Google อาจประมวลผลตัวระบุอุปกรณ์ ข้อมูลการใช้งาน ข้อมูลทางเทคนิค และสัญญาณการโฆษณาตามนโยบายของ Google และการตั้งค่าโฆษณาของผู้ใช้

### 3.10 ข้อมูลจาก API ของตลาดภายนอก

แอปสามารถปรึกษา API ภายนอกเกี่ยวกับอัตราแลกเปลี่ยน ราคา ดัชนี หรือข้อมูลตลาด โดยทั่วไปข้อความค้นหาเหล่านี้จะเกี่ยวข้องกับพารามิเตอร์ต่างๆ เช่น สกุลเงิน สินทรัพย์ ประเทศ หรือสัญลักษณ์ที่กำลังค้นหา ไม่จำเป็นต้องส่งข้อมูลทางการเงินส่วนบุคคลที่ลงทะเบียนของคุณเพื่อรับข้อมูลนี้

- การปฏิบัติตามภาระผูกพันทางกฎหมายหรือข้อบังคับเมื่อจำเป็น
- การใช้สิทธิเป็นประจำในกระบวนการพิจารณาคดีปกครอง ตุลาการ หรือวิสามัญฆาตกรรม

คุณสามารถเพิกถอนความยินยอมเพิ่มเติมได้ในการตั้งค่าแอปหรือระบบปฏิบัติการ แต่การดำเนินการนี้อาจจำกัดฟังก์ชันการทำงาน

เราอาจแบ่งปันหรืออนุญาตให้มีการประมวลผลข้อมูลโดยผู้ให้บริการที่จำเป็นสำหรับการทำงานของแอป รวมถึง:

- **Google Firebase**: Firebase Authentication, Cloud Firestore, Firebase Cloud Messaging, Firebase Analytics และ Firebase Crashlytics
- **การลงชื่อเข้าใช้ Google / การลงชื่อเข้าใช้ Apple**: เข้าสู่ระบบผ่านบัญชีบุคคลที่สามเมื่อคุณเลือก
- **Google Play Billing**: การประมวลผลการสมัครสมาชิก การซื้อ การยกเลิก การกู้คืน และสถานะของ Premium
- **Google AdMob**: การแสดงและการวัดผลโฆษณา
- **API ตลาด การแลกเปลี่ยนเงินตราต่างประเทศ หรือใบเสนอราคา**: การให้คำปรึกษาเกี่ยวกับข้อมูลทางการเงินสาธารณะหรือตลาดที่ใช้ในแอป
- **Android บริการระบบ**: การอนุญาต การแจ้งเตือน และทรัพยากรของอุปกรณ์

บุคคลที่สามเหล่านี้อาจประมวลผลข้อมูลตามเงื่อนไข นโยบาย และการตั้งค่าของตนเอง

## 7. การโอนเงินระหว่างประเทศ

บริการต่างๆ เช่น Firebase, Google Play, AdMob และผู้ให้บริการอื่นๆ อาจประมวลผลข้อมูลบนเซิร์ฟเวอร์ที่ตั้งอยู่นอกบราซิล โดยการใช้แอป คุณรับทราบว่าการประมวลผลระหว่างประเทศนี้อาจเกิดขึ้นได้ ขึ้นอยู่กับกฎการคุ้มครองข้อมูลที่เกี่ยวข้องและข้อกำหนดของผู้ให้บริการ

## 8. การเก็บรักษา

เราเก็บข้อมูลไว้นานเท่าที่จำเป็นเพื่อส่งใบสมัคร บรรลุวัตถุประสงค์ที่อธิบายไว้ในนโยบายนี้ ปฏิบัติตามภาระผูกพันทางกฎหมาย แก้ไขข้อพิพาท ป้องกันการฉ้อโกง และรักษาความปลอดภัย

โดยทั่วไป:

- ข้อมูลบัญชีและข้อมูลทางการเงินในระบบคลาวด์จะยังคงอยู่ตราบเท่าที่มีบัญชีอยู่หรือจนกว่าจะมีคำขอลบที่ถูกต้อง
- ข้อมูลในเครื่องอาจยังคงอยู่ในอุปกรณ์จนกว่าคุณจะลบ ล้างข้อมูลแอป หรือถอนการติดตั้งแอป
- ข้อมูลการวินิจฉัย การวิเคราะห์ และบันทึกสามารถเก็บไว้ตามระยะเวลาที่ผู้ให้บริการกำหนด
- ข้อมูลการซื้ออาจยังคงอยู่ใน Google Play ตามกฎด้านภาษี การค้า และร้านค้า
- ข้อมูลที่ไม่ระบุชื่อหรือรวมอาจถูกเก็บไว้โดยไม่ระบุตัวคุณ

## 9. ความปลอดภัย

เราใช้มาตรการทางเทคนิคและองค์กรที่เหมาะสมเพื่อปกป้องข้อมูล รวมถึง:

- การรับรองความถูกต้องโดย Firebase
- กฎการเข้าถึงในฐานข้อมูลเพื่อจำกัดข้อมูลให้กับผู้ใช้ที่เกี่ยวข้อง
- การสื่อสารที่เข้ารหัสโดย SDK และ API ที่ใช้
- การใช้สิทธิ์ Android เฉพาะเมื่อเกี่ยวข้องกับทรัพยากรของแอป
- กรองเมื่อตรวจสอบการแจ้งเตือนเพื่อหลีกเลี่ยงรหัสความปลอดภัย OTP รหัสผ่าน และข้อความการตรวจสอบสิทธิ์
- การวินิจฉัยข้อผิดพลาดเพื่อแก้ไขข้อผิดพลาดและปรับปรุงเสถียรภาพ

ไม่มีระบบใดที่ปลอดภัย 100% คุณควรปกป้องอุปกรณ์ของคุณ ใช้การล็อกหน้าจอ เก็บรหัสผ่านที่รัดกุม หลีกเลี่ยงการแชร์บัญชีของคุณ และอัปเดตแอปเมื่อมีการเปิดตัวเวอร์ชันใหม่
